"Monte Adentro" is a narrative deduction game set in an Andean Colombian
village, Unity 6.3 LTS, PC, single-player. Central creative pillar: mythic/
supernatural elements are never visually flagged as "different" — they only
become legible to the player after the player has learned to interpret the
world (see Art Bible Section 1, Principle 1: "el indicio se integra, nunca se
anuncia").

Art Bible is being drafted section-by-section with incremental
approval (per the project's standard collaborative protocol). As of
2026-07-15:
- Section 1 (tone/mythic-ambiguity principles) — approved.
- Section 2 (5 time-of-day narrative states tied to the village's real clock:
  cotidiano/mediodía → investigación/tarde → preparación/crepúsculo →
  confrontación/noche → después-del-caso/amanecer bifurcado
  favorable-vs-desfavorable, never coded as "victory/defeat" light) — approved.
- Section 3 (two-tier silhouette: ambiguous world vs. clear investigator's
  notebook; geometry gradient orthogonal-imperfect village → organic monte →
  ambiguous liminal zone which is the natural habitat of clues; hard rule that
  shape may signal interactability but never "this is a clue") — approved.
- Section 4 (Color System) — drafted 2026-07-15, pending user approval.

**Why this matters:** Later sections (lighting, typography, VFX, SFX/music
direction) must stay consistent with these approved principles, especially
the "never flag clues directly" rule and the day/night state structure. See
[[art-bible-editorial-rule]] for the hard constraint added alongside Section 4.

**How to apply:** Before drafting further Art Bible or audio-direction
sections, cross-check against this approved-sections summary rather than
re-deriving tone from scratch. Verify current approval status in the actual
Art Bible file before relying on this snapshot, since it will go stale.
